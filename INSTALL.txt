======================================================
  Cloud Instance Verifier - Installation Guide
======================================================

---------------------------------------------
Prerequisites:
  - Python 3.10 or higher  →  https://www.python.org/downloads/
    IMPORTANT: during install, check "Add Python to PATH"

Steps:
  1. Copy this entire folder to your machine
  2. Double-click  install.bat   (first time only, installs dependencies)
  3. Double-click  run.bat       (every time you want to use the app)
  4. The browser opens automatically at http://localhost:5000

How to use:
  1. Upload your Excel file (drag & drop or click to select)
  2. Enter the column letters:
       - Links column   → letter of the column with AWS Calculator URLs
       - Customer column → letter of the column with company/customer names
       - Header row     → row number of the column headers (usually 6)
  3. Choose output type: By company / Grand total / Detailed
  4. Click Process
  5. Download the result Excel when done
